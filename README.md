# Modus

A Pomodoro focus timer app inspired by [pomofocus.io](https://pomofocus.io). This project builds upon the original idea with additional features and customizations.

## Features

- **Pomodoro Timer** (25 min default)
- **Short Break** (5 min default)
- **Long Break** (15 min default)
- **Task List** (add, check off, reorder, delete)
- **Clean and Minimal UI**
- **LocalStorage Persistence** (tasks + timer settings)
- **Settings Modal** with customizable durations and themes

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## Tech Stack

- Vite + React + TypeScript
- TailwindCSS
- Zustand (state management)
